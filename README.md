# JavaScript – Javier Montilla Arias' Practice Repository
 
This repository contains JavaScript exercises and projects developed by **Javier Montilla Arias** following the **Zero to Mastery (ZTM)** course guide. It covers foundational topics in modern JavaScript (ES6+) with a hands-on approach through practical examples and mini-projects.
 
---
 
## Repository Structure
 
### Basics
Core JavaScript concepts and syntax exercises, including:
- Variables (`var`, `let`, `const`) and data types
- Conditionals (`if/else`, `switch`) and comparison operators
- Loops (`for`, `while`, `for...of`) for iteration
- Functions, arrow functions, and scope
- Arrays and objects fundamentals
### DOM
Exercises focused on interacting with the Document Object Model:
- Selecting and modifying HTML elements dynamically
- Changing styles and CSS classes via JavaScript
- Creating and removing elements from the page
- Traversing the DOM tree (parent, child, and sibling nodes)
### Events
Hands-on practice with browser event handling:
- Click, keydown, keyup, and input events
- Event listeners and the event object
- Form validation and user input handling
- Preventing default behaviors and event propagation
### Projects
Real-world mini-projects that bring together all of the above concepts:
 
| Project | Description |
|---|---|
| **Quote Generator** | Fetches and displays random quotes from an external API, with a Twitter share button |
| **Infinite Scroll** | Loads new images automatically as the user scrolls down the page, using the Unsplash API |
| **Light / Dark Mode** | Toggles between light and dark themes with smooth transitions, persisting the user's preference |
| **Picture in Picture** | Uses the browser's Picture-in-Picture API to display a video in a floating window |
